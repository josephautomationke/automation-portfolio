# AI Lead Generation (Telegram)

## What this project does
This automation collects leads from Telegram, processes them, and helps identify high-quality potential customers.

## How it works
1. User sends message in Telegram
2. n8n receives the message
3. Data is cleaned and structured
4. AI or rules classify the lead quality
5. Lead is stored or sent to notification system

## Tools used
- n8n
- Telegram Bot API
- Google Sheets (optional)
- AI classification (optional)

## Result
This system reduces manual lead handling and helps businesses respond faster to potential customers.
